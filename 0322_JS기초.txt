  함수는    정의가    있고       호출이 있다.

4. JS confirm 경고창의 기능
   => 화면에서 "확인", "취소" 버튼 2개들 표시한다.
       1) "확인" 버튼을 눌렀을 경우
           => confirm 함수는 실행이 종료되지만
                true 라는 값을 생성하여 브라우저에 남긴다. 
       2) "취소" 버튼을 눌렀을 경우
           => confirm 함수는 실행이 종료되지만
                false 라는 값을 생성하여 브라우저에 남긴다. 

        참고. 함수가 실행 후 종료되면서, 브라우저나 JS
                및 어떤 대상이라도 값을 남기는 것을
                반환값 또는 반환이라고 한다.
                (return value, 리턴 밸류, 리턴 값)  

5. 반환값을 사용하여
    선택적인 분기를 만들 수 있다.

     if ( B ) {
        B가 true일 때 실행할 코드;
     } else {
        B가 false일 때 실행할 코드;
     }

    B에는 true 또는 false를 따져볼수있는 값이나 식을
    기재하고 명칭을 "조건식"이라고 한다.
                        true, false를 따져볼 수 있는 식
    
     참고1. 조건식이 true가 된다면
             주어진 분기에 관련된 식 if문은
             아래와 같이 형태가 변형된다.
     if ( true ) {
        B가 true일 때 실행할 코드;
     } else {
        B가 false일 때 실행할 코드;
     }       
     => 
         B가 true일 때 실행할 코드;

     참고2. 조건식이 false가 된다면
             주어진 분기에 관련된 식 if문은
             아래와 같이 형태가 변형된다.
     if ( false ) {
        B가 true일 때 실행할 코드;
     } else {
        B가 false일 때 실행할 코드;
     }       
     => 
         B가 false일 때 실행할 코드;
